# Studio

Studio lets you build NativeBlade apps with a clean, high-level syntax. It is
driven over the Model Context Protocol (MCP) and follows MCP standards, so you
bring **your own AI** to generate and shape the app while Studio provides the
syntax and the framework-aware context.

- Repository: [github.com/NativeBlade/studio](https://github.com/NativeBlade/studio){target="_blank" rel="noopener"}

## How it fits

- **Clean syntax:** you describe the app at a high level instead of wiring every
  detail by hand.
- **MCP standard:** Studio exposes the framework over MCP, so any MCP-capable
  assistant can drive it.
- **Your own AI:** you are not tied to a specific model or provider. Use the
  assistant you already work with.
