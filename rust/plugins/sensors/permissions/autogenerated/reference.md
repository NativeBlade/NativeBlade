## Default Permission

Default permission set for the NativeBlade sensors plugin.
Allows one-shot sensor reads and starting/stopping throttled watches.

#### This default permission set includes the following:

- `allow-is-available`
- `allow-read-sensor`
- `allow-watch-sensor`
- `allow-stop-sensor`
