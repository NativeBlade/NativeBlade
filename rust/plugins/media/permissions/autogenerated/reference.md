## Default Permission

Default permission set for the NativeBlade media plugin.
Allows opening the camera and the gallery, querying and requesting
the OS-level camera/photos permissions, and reading bytes from an
asset URL produced by a previous pick.

#### This default permission set includes the following:

- `allow-pick-from-camera`
- `allow-pick-from-gallery`
- `allow-pick-video`
- `allow-check-permissions`
- `allow-request-permissions`
- `allow-read-asset`
