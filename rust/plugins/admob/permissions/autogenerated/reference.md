## Default Permission

Default permission set for the NativeBlade admob plugin.
Allows requesting ad consent and showing rewarded, interstitial and banner ads.

#### This default permission set includes the following:

- `allow-request-consent`
- `allow-show-rewarded`
- `allow-show-interstitial`
- `allow-show-banner`
- `allow-hide-banner`
